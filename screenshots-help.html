Solus Marketplace - UI/UX

Это инструмент для создания скриншотов UI/UX дизайна Telegram мини-приложения "Solus Marketplace".

Примечание:

Для правильного отображения всех экранов необходимо добавить соответствующие изображения в папку img/, как указано в README.md

Для корректного отображения на iPhone, используйте режим эмуляции мобильного устройства в инструментах разработчика (DevTools).

Как делать скриншоты:

  1. Откройте нужный экран, используя ссылки ниже
  2. Используйте DevTools для эмуляции iPhone (F12 → кнопка "Toggle device toolbar")
  3. Сделайте скриншот (на Mac: Cmd+Shift+4, на Windows: Win+Shift+S)
  4. Импортируйте скриншоты в Figma для создания прототипа

Основные экраны

  • Профиль (главный экран)
  • Маркетплейс
  • Информация

Дополнительные экраны

  • Корзина
  • Детали товара
  • Перевод коинов

Админ-панель

  • Главный экран админ-панели
  • Управление задачами
  • Управление пользователями
  • Управление маркетплейсом
  • Автоматические начисления

Модальные окна

Для отображения модальных окон необходимо нажать на соответствующий элемент:

  • Информация об уровнях (нажмите на "Бронзовий")
  • Добавление коинов (нажмите на "+")
  • Редактирование товара (нажмите на ✏️)
